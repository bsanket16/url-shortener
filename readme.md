## Deployment

This app is deployed using heroku.

```bash
https://surll.herokuapp.com
```

## Installation

Use the package manager npm / yarn to install.

```bash
npm install
yarn install
```
## Development & Production

Use one of the following as required.

```bash
npm dev
npm start
```
